This is a chatgpt Translator. 
Translates any language messages into English.

Request

curl --location 'http://localhost:8080/api/chatgpt/ask' \
--header 'Content-Type: application/json' \
--form 'prompt="கற்றலின், கேட்டலே நன்று"'
